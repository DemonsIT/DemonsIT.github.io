1. 准备工作

  • Git

    • 检测是否安装成功:
    • 配置用户账户和邮箱
    • 生成SSH密钥文件
    • 检测GitHub公钥是否配置成功
    • 关联远程仓库
    • 向远程仓库上的master主干推送本地分支

  • GitHub

    • 新建仓库,仓库名称必须是: username.github.io
    • 配置全局SSH
      • Git生成的本地SSH在C:\Users\dell\.ssh路径下,将其中的id_rsa.pub中的内容复制至GitHub的SSH中

  • Node.js(安装后包含npm)

    • 检测是否安装成功:

  • Hexo

    • 首先新建一个文件夹Blog,用于搭建环境
    • 在Blog中打开Git Bash Here,安装Hexo
    • 命名和初始化首个博客(依次单步执行)

  • 执行,启动服务(本地测试)
