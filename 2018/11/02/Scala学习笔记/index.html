4.1 声明变量

  • 1、val

  • 2、var

  • scala中在声明变量的时候，可以不给出变量的类型，可以通过scala编译器自动帮我们进行类型推断。

  • 也可以具体给出变量的类型

    • val e:String=”itcast”
    • 这里是变量名在前，类型在后，跟java正好相反。

4.6 定义方法和函数

  • 1、定义方法

    • def m1(x:Int,y:Int):Int=x+y

      /5%E3%80%81scala%E8%AF%BE%E7%A8%8B%E8%B5%84%E6%96%99/scala_day01/method.png)

    • 调用方法m1(1,2)

    • 声明一个方法的时候，返回值类型可以省略不写，但是递归必须要手动指定返回值类型

  • 2、定义函数

    • val f1=(x:Int,y:Int) => x+y
    • f1(1,2)

  • 3、方法和函数区别

    • 函数
    • 函数在scala中它是一个对象，具有该对象中的一些方法
    • 当前这个函数有N个输入参数，该函数就继承自FunctionN类
    • 方法
      • 方法就是对象中的方法，它是不具备函数这种特性。

  • 4、方法转换成函数

    • 使用方法名+下划线转换成函数
      • val f1=m1 _

6、映射

  • 就是一个map集合

  • 定义不可变的map集合

    • val map=Map(“键” -> “值” ,”键” -> “值”)

    • val map=Map((“键”,”值”)，(“键”,”值”))

  • 获取value

    • 值=map(键)
    • map.getOrElse(键，默认值)
      • 如果key存在就获取真实的值，如果不存在就给一个默认值，避免在不确定map中是否存在这个key的情况下报错

  • 定义可变的map集合

    • import scala.collection.mutable.HashMap

    • val hmap=HashMap(“zhangsan” -> 30,”lisi” -> 40)

7、元组

  • 1、元组就是把不同类型的值通过小括号聚在一起就组成一个元组
  • 2、定义元组
    • val t=(100,3.14,”itcast”)
  • 3、获取元组中的元素
    • t._1
    • t._2
    • t._3
    • 注意：从1开始，不是从0开始

8、List

  • 1、不可变的list（immutable）

    • val list=List(1,2,3,4)

  • 2、可变的list（mutable）

9、Set

  • 1、不可变的set集合（immutable）

    • val set=Set(1,2,3,4)

  • 2、可变的set集合（mutable）

    • import scala.collection.mutable.HashSet

    • val hset=HashSet(1,2,3)
